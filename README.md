# 🤖 Mini Jarvis - AI Voice Assistant

Mini Jarvis is a Python-based virtual assistant inspired by Iron Man's JARVIS. It can perform various tasks such as opening applications, searching the web, telling the time, answering questions, and interacting through voice commands.

---

## 🚀 Features

✅ Voice Recognition

✅ Text-to-Speech Responses

✅ Open Websites (Google, YouTube, GitHub, etc.)

✅ Search Information Online

✅ Tell Current Time & Date

✅ Launch Applications

✅ Perform Basic Calculations

✅ Personalized Greetings

✅ Easy to Extend with New Commands

---

## 🛠️ Tech Stack

- Python
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia API
- Datetime
- Webbrowser
- OS Module
